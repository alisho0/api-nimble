# Nimble Gravity – Junior Fullstack Developer Challenge

Mini aplicación en React desarrollada como parte del proceso de selección para el rol de Junior Fullstack Developer.

La aplicación:

- Obtiene los datos del candidato mediante su email.
- Consulta la lista de posiciones abiertas.
- Muestra las posiciones en una interfaz simple.
- Permite ingresar la URL del repositorio de GitHub.
- Envía la postulación a la API correspondiente.
